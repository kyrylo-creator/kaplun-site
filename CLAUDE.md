## Архітектура

- **Eleventy (v3)** — генератор статичного сайту.
- **Decap CMS** — щоб психолог сам публікував статті (адмінка `/admin`, бекенд **github**, вхід через GitHub OAuth).
- Хостинг — **Netlify** (build з `netlify.toml`).
- Контактна форма — **Netlify Forms** (на сторінці контактів).

```
src/
  index.html          головна (готова верстка, проходить як passthrough)
  styles.css, script.js
  assets/             фото, дипломи; assets/uploads/ — картинки зі статей (Decap)
  blog.njk            список статей  → /blog.html
  posts/
    trivoga.md        приклад статті
    posts.json        спільні налаштування статей (layout/tags/permalink)
  _includes/
    base.njk          каркас (шапка/футер)
    post.njk          шаблон статті
  admin/
    index.html        Decap CMS
    config.yml        колекція «posts», бекенд github (заміни OWNER/REPO!)
.eleventy.js          конфіг генератора
netlify.toml          build: npm run build, publish: _site
SETUP.md              ПОВНА покрокова інструкція деплою + OAuth
```

## Команди

```bash
npm install
npm run dev      # http://localhost:8080
npm run build    # збірка в _site/
```
(У Cowork-пісочниці збірку в `_site` блокував захист видалення — тестували через
`--output=/tmp/...`. На звичайному Mac/у терміналі цього обмеження немає, `npm run build` працює як треба.)
