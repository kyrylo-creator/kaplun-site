# Як публікувати статті в блог — інструкція для Саші

Привіт! Нижче — крок за кроком, як отримати доступ до адмінки і
опублікувати першу статтю. Це треба пройти **тільки один раз**,
далі — просто заходиш і пишеш.

Будь ласка, виконуй послідовно. Якщо щось не виходить — пиши Кирилу
скрін і опис, де застряг.

---

## Крок 1. Зареєструватися на GitHub (≈3 хв)

GitHub — це майданчик, де лежить твій сайт. Тобі потрібен **безкоштовний**
акаунт, щоб ним користуватися. Жодних оплат, нічого складного.

1. Зайди на **https://github.com/signup**
2. Введи **свою електронну пошту** → Continue
3. Придумай **пароль** (не менше 8 символів, бажано довший — збережи його
   в нотатках) → Continue
4. Придумай **username** (логін) — це твоє публічне ім'я на GitHub.
   Можеш узяти просто `sashkopsy` або `oleksandr-kaplun` — щось схоже.
   Якщо ім'я вже зайняте — GitHub запропонує варіанти. → Continue
5. На питання «Are you interested in receiving emails…» — як хочеш
   (можеш зняти галочку) → Continue
6. **Verify your account** — пройди коротку капчу
7. **Create account**
8. На пошту прийде **код підтвердження** (6 цифр) — введи його на сайті
9. Далі GitHub запитає купу всякого («What's your role», «How many people
   on your team» тощо) — це необов'язково. Можеш вибирати будь-що, або
   натиснути **Skip personalization** внизу.
10. Готово — ти на головній сторінці GitHub.

> **Збережи свій логін GitHub** і **пароль** у надійному місці.
> Напиши Кирилу у Telegram свій **GitHub-логін** — він додасть тебе.

## Крок 2. Прийняти запрошення стати співавтором

Після того як ти напишеш Кирилу свій логін, він додасть тебе до сайту як
співавтора (collaborator). Тобі прийде запрошення.

1. **Перевір пошту** (ту, на яку реєструвався в GitHub) — буде лист
   від `noreply@github.com` із темою на кшталт
   _"\[GitHub] @kyrylo-creator invited you to kaplun-site"_
2. У листі натисни велику кнопку **View invitation**
3. Відкриється сторінка GitHub — натисни зелену кнопку
   **Accept invitation**
4. Готово. Можеш просто закрити вкладку.

---

## Крок 3. Перший вхід в адмінку (≈1 хв)

Тепер ти можеш заходити в адмінку сайту й публікувати статті.

1. Відкрий: **https://sashkopsy.com/admin**
2. Побачиш екран із кнопкою **Login with GitHub** — натисни її
3. Відкриється віконце GitHub зі словами _"sashkopsy.com admin wants to
   access your account"_ → натисни **Authorize**
4. Тебе перекине назад на адмінку сайту, і ти побачиш список статей
   (зараз там одна-дві приклади).

**Це треба зробити лише раз.** Далі заходитимеш просто
`sashkopsy.com/admin` → одразу всередині.

## Крок 4. Опублікувати першу статтю

1. У лівому меню адмінки натисни **Статті блогу**
2. Зверху справа — кнопка **New Стаття** (синя). Натисни.
3. Заповни поля:
   - **Мова статті** — UK / EN / RU. Стаття автоматично потрапить у
     блог обраної мови:
     - UK → `sashkopsy.com/uk/blog/`
     - EN → `sashkopsy.com/blog/`
     - RU → `sashkopsy.com/ru/blog/`
     Якщо хочеш ту саму статтю іншою мовою — створи нову, постав
     іншу мову й скопіюй текст.
   - **Заголовок** — назва статті
   - **Дата** — дата публікації (за замовчуванням сьогодні)
   - **Категорія / підзаголовок** — напр. _«Тривога · 5 хв читання»_
     або _«Anxiety · 5 min read»_
     (відображається маленьким сірим над заголовком)
   - **Короткий опис** — 1–2 речення, які видно на картці в блозі та які
     використовуються для пошуковиків. Має «продати» клік.
   - **Опис для пошуковиків** — можна не заповнювати, тоді візьметься
     попереднє поле
   - **Текст статті** — тут пишеш саму статтю
4. У редакторі текста є кнопки: **B** жирний, **I** курсив, заголовки
   `H1/H2/H3`, списки, цитата, картинка, посилання. Працює як простий
   Word. У правому верхньому куті є перемикач **Rich Text / Markdown** —
   просто залиш Rich Text.
5. Щоб **вставити картинку**: натисни значок зображення → **Choose an
   image** → можеш завантажити з комп'ютера або вибрати з уже
   завантажених. Картинка автоматично з'явиться в потрібному місці.
6. Коли стаття готова — справа вгорі натисни:
   - **Save** — збереже як чернетку (Draft), на сайті не з'явиться
   - **Publish → Publish now** — публікує негайно
7. **Через 1–2 хвилини** стаття з'явиться на `sashkopsy.com/blog.html`.
   Це нормальна затримка — сайт пересоберется автоматично.

---

## Кілька корисних дрібниць

- **Чернетки.** Поки тиснеш Save, а не Publish — стаття є в адмінці, але
  на сайті не видно. Це зручно, щоб написати половину і повернутися
  завтра.
- **Редагувати опубліковану.** Зайди в список статей → клікни потрібну
  → виправ → Publish. Через хвилину зміни підхопляться.
- **Видалити статтю.** Усередині статті — три крапки справа вгорі →
  **Delete entry**. Видалиться з сайту через хвилину.
- **Картинки.** Усі завантажені картинки зберігаються в розділі **Media**
  (значок ↑ зліва внизу). Звідти можна потім перевикористати.
- **Помилка «Login failed».** Закрий браузер, відкрий заново, заходь
  знову. Зазвичай це вирішує.
